# backend_php_3sem

О проекте
Notebook — это веб-приложение для хранения и управления контактной информацией, разработанное в рамках курсовой работы по дисциплине «Серверная веб-разработка». Приложение представляет собой записную книжку с возможностью просмотра, добавления, редактирования и удаления контактов.

✨ Основные возможности
✅ Просмотр контактов — табличное отображение с пагинацией (10 записей на страницу)

🔄 Сортировка — по дате добавления, фамилии и дате рождения

➕ Добавление записей — форма с валидацией обязательных полей

✏️ Редактирование — выбор контакта из списка и изменение данных

🗑️ Удаление — с подтверждением и уведомлением об успехе

🎨 Адаптивный дизайн — корректное отображение на всех устройствах

🛡️ Безопасность — защита от SQL-инъекций и XSS-атак

🏗️ Архитектура
Проект построен по модульному принципу с единой точкой входа:

text
notebook/
├── index.php # Единственная точка входа (маршрутизатор)
├── config.php # Конфигурация и подключение к БД
├── menu.php # Генерация навигационного меню
├── viewer.php # Просмотр контактов с пагинацией
├── add.php # Добавление нового контакта
├── edit.php # Редактирование контакта
├── delete.php # Удаление контакта
├── assets/
│ └── css/
│ └── style.css # Стили оформления
└── README.md # Этот файл

🚀 Быстрый старт
Требования
PHP 7.4 или выше

MySQL 5.7 или выше

Веб-сервер (Apache / Nginx / OpenServer / XAMPP)

Установка
Клонируйте репозиторий

```bash
git clone https://github.com/i1e7pl3/notebook.git
```
cd notebook


Создайте базу данных

```sql
CREATE DATABASE notebook CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

Импортируйте структуру таблицы

```bash
mysql -u root -p notebook < database.sql
```
или выполните SQL-код из файла contacts.sql в phpMyAdmin

Настройте подключение к БД
Откройте config.php и при необходимости измените параметры:

```php
$host = '127.0.0.1';    // или 'localhost'
$db = 'notebook'; // имя базы данных
$user = 'root';         // пользователь MySQL
$pass = ''; // пароль
```

Запустите приложение
Поместите проект в директорию веб-сервера (например, htdocs для XAMPP) и откройте в браузере:

text
http://localhost/notebook/
